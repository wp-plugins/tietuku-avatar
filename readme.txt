Use Tietuku Avatar service to replace Gravatar.

== Description ==

This plugin replaces build-in Gravatar with Tietuku Avatar Service.

== Installation ==

1. Upload `tietuku_avatar.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
